## 🧩 1️⃣ 迭代器的基本概念

**迭代器（Iterator）** 是一种“泛化的指针”。
 它是 STL 中用来**访问容器元素**的统一接口。

你可以把它理解成：

> “能在不同容器中移动的指针”。

------

### 📦 举例说明：

```
std::vector<int> v = {1, 2, 3};

auto it = v.begin();   // 指向第一个元素
std::cout << *it;      // 输出 1

++it;
std::cout << *it;      // 输出 2
```

上面：

- `begin()` 返回迭代器，指向容器首元素
- `end()` 指向“尾后”位置（不指向实际元素）
- `*it` 解引用访问元素
- `++it` 移动到下一个元素

------

### 📚 STL 算法通用接口

所有 STL 算法都是通过迭代器来访问容器元素的，比如：

```
#include <algorithm>
std::sort(v.begin(), v.end());
```

这条语句不关心 `v` 是 `vector` 还是 `deque`，
 只要它支持随机访问迭代器即可。

这就是 **“容器与算法解耦”** 的核心设计思想。

------

## 🧠 2️⃣ 迭代器的分类（五种）

STL 根据**迭代器的能力**，定义了五种类型：

| 分类                       | 可操作范围   | 支持操作              | 示例容器             |
| -------------------------- | ------------ | --------------------- | -------------------- |
| **Input Iterator**         | 只能读，单向 | `++`, `*`             | `istream_iterator`   |
| **Output Iterator**        | 只能写，单向 | `++`, `*=`            | `ostream_iterator`   |
| **Forward Iterator**       | 可读写，单向 | `++`, `*`, 可多次遍历 | `forward_list`       |
| **Bidirectional Iterator** | 可读写，双向 | `++`, `--`, `*`       | `list`, `set`, `map` |
| **Random Access Iterator** | 可随机访问   | `+`, `-`, `[]`, `<`   | `vector`, `deque`    |

| 容器名               | 迭代器类型                               |
| -------------------- | ---------------------------------------- |
| `vector`             | 随机访问迭代器（random_access_iterator） |
| `deque`              | 随机访问迭代器（random_access_iterator） |
| `list`               | 双向迭代器（bidirectional_iterator）     |
| `forward_list`       | 单向迭代器（forward_iterator）           |
| `array`              | 随机访问迭代器（random_access_iterator） |
| `string`             | 随机访问迭代器（random_access_iterator） |
| `set`                | 双向迭代器（bidirectional_iterator）     |
| `multiset`           | 双向迭代器（bidirectional_iterator）     |
| `map`                | 双向迭代器（bidirectional_iterator）     |
| `multimap`           | 双向迭代器（bidirectional_iterator）     |
| `unordered_set`      | 前向迭代器（forward_iterator）           |
| `unordered_multiset` | 前向迭代器（forward_iterator）           |
| `unordered_map`      | 前向迭代器（forward_iterator）           |
| `unordered_multimap` | 前向迭代器（forward_iterator）           |

## ✅ 总结口诀

| 容器类型             | 迭代器稳定性                     |
| -------------------- | -------------------------------- |
| `vector`, `string`   | 最不稳定，扩容全挂               |
| `deque`              | 比 vector 稳，但首尾插删可能全挂 |
| `list`, `map`, `set` | 非常稳定，只删自己挂             |
| `unordered_*`        | rehash 全挂                      |
| `array`              | 永不失效                         |
