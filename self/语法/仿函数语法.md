**仿函数语法 = 重载 operator() 的类对象**

1. 定义类

```
struct A { R operator()(Args...) const { ... } };
```

1. 创建对象

```
A a;
```

1. 像函数一样调用

```
a(x, y);
```

1. 能保存状态（成员变量）
2. 能 inline，性能高于函数指针
3. 常用于 STL 算法：

```
sort(v.begin(), v.end(), A());
for_each(v.begin(), v.end(), A());
count_if(v.begin(), v.end(), A());
```
