静态成员对象和静态成员函数

## 静态成员（`static` member）

在类中使用 `static` 修饰的成员变量或成员函数，称为**静态成员**。
 它们**属于类本身**，而不是某个具体对象。

## 🧩 一、静态成员变量（`static` 成员变量）

### ✅ 1. 定义方式

静态成员变量属于 **类本身**，而不是属于类的某个对象。

```
class MyClass {
public:
    static int count;  // 声明静态成员变量
};
```

### ✅ 2. 定义与初始化（类外）

必须 **在类外定义一次**，否则会出现链接错误。

```
int MyClass::count = 0;  // 定义并初始化
```

> ⚠️ 注意：
>
> - 静态成员变量 **在所有对象间共享**，只有一份存储空间。
> - 静态成员变量可以在对象创建前访问。

------

### ✅ 3. 使用方式

- 通过 **类名** 或 **对象** 都可以访问。

```
MyClass::count = 5;  // 推荐方式
MyClass obj1;
obj1.count = 10;     // 语法允许，但效果一样
```

### ✅ 4. 生命周期

静态成员变量的生命周期与程序一致（和全局变量类似）。

## ⚙️ 二、静态成员函数（`static` 成员函数）

### ✅ 1. 定义方式

静态成员函数同样属于类本身，而不是某个对象：

```
class MyClass {
public:
    static void showCount();
};

void MyClass::showCount() {
    std::cout << "count = " << count << std::endl;
}
```

> 静态函数不需要对象实例即可调用。

------

### ✅ 2. 调用方式

```
MyClass::showCount();  // 推荐
MyClass obj;
obj.showCount();       // 语法允许
```

------

### ✅ 3. 限制

静态成员函数 **不能访问非静态成员变量或函数**，因为它没有 `this` 指针。

✅ 可以访问：

- 静态成员变量；
- 其他静态成员函数。

❌ 不能访问：

- 非静态成员变量；
- 非静态成员函数。
