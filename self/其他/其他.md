## 一、位域是什么（定义级别）

**位域是 C 语言里一种“结构体成员声明方式”**，用于声明：

> **一个成员只占用若干 bit，而不是整个 int / char**
>
> struct foo {
>  unsigned int a : 1;
>  unsigned int b : 3;
>  unsigned int c : 4;
>
> };
>
> 

`asmlinkage` 是 Linux 内核里的一个 **函数调用约定（calling convention）宏**，它主要影响函数 **参数是从哪里取的**。

`asmlinkage` 就是告诉编译器：

> 这个函数的参数 **都从栈上取**，不要假定参数在寄存器里。

`size_t` 是一个“能表示平台上最大对象大小 / 内存大小”的无符号整数类型。

`noexcept` 用在函数声明或定义前面，意思是：**告诉编译器这个函数不会抛异常**

`volatile` 是一个类型修饰符，意思是：**告诉编译器：这个变量的值可能随时被“外部因素”改变，不要对它做优化假设。**
